# experimental-projects
A collection of small applications, experiments, and backend projects built for learning, testing ideas, and improving development skills.

This repository serves as a public playground for experimenting with different technologies, architectures, and patterns.


# Projects:
1-) simple-voting-app: is a simple election simulation application.

    It was designed to simulate voting processes for small-scale elections, such as:

    * CIPA (Internal Commission for Accident Prevention) elections
    * Union elections
    * Small organizational voting processes

    No front has been used.

2-) cnab_reader: Code to read any kind of CNABs documents

    It extracts data based on predefined positional mappings provided in the configuration.
    The solution is backend-focused, with no frontend implementation.
